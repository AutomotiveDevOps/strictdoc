Getting started
$$$$$$$$$$$$$$$

Installing Saturn as a Pip package
=====================================

.. code-block:: text

    pip install saturn

Installing Saturn from GitHub (development)
==============================================

.. code-block:: text

    git clone git@github.com:stanislaw/saturn.git && cd saturn
    poetry install
    poetry run saturn

Hello world
===========

.. code-block:: text

    git clone git@github.com:stanislaw/saturn.git && cd saturn
    saturn export docs/
